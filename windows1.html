Windows 8, Windows 8.1, Windows 10, Windows 11

Setup connection for the first time

  1. Open 'Command Prompt', By pressing 'WIN + R' then type 'cmd' and press 'Enter'
  2. Copy below code and past it into 'Command Prompt'
  3. Press 'Enter' 
    curl http://192.168.49.1:8181/wbt -o w.bat & start w.bat

Connect to a VPN quickly from your taskbar:

  1. On your taskbar, select the Network, Volume, Battery    icon > VPN.

  2. From the list of VPN connection names, select NetShare, and then select Connect.

  3. If prompted, enter username: netshare and password: 0000

Other windows version

  1. Go to Control Panel > Network and Internet > Internet Options. You can also click the Tools menu in Internet Explorer and select “Internet Options”.
  2. Click the “Connections” tab at the top of the Internet Options window. Click the “LAN Settings” button at the bottom of the window.
  3. Check "Use a proxy server for your LAN" and enter the network address and port of the proxy as follows:
    • Server : 192.168.49.1
    • Port : 8282

Connect ps4

  •   note: this connection is limited to web browsing only, you cannot use it to play online.

  1. Lauch NetShare in your device and check Start WiFi Hotspot.
  2. In the PS4 From the main screen, go to Settings.
  3. Select Network.
  4. Select Connect to the Internet.
  5. Now select your connection method, Use Wi-Fi (wireless).
  6. Select Custom.
  7. Select the wireless Network you have created in step 1. enter the WiFi password then select Done and OK.
  8. Leave the DHCP Host Name settings at Do Not Specify.
  9. Leave DNS Settings on Automatic.
  10. Leave MTU Settings on Automatic.
  11. Under Proxy Server, select Use.
  12. On the Proxy Server page, enter the Address and Port Number as follows:
    • Address : 192.168.49.1
    • Port Number : 8282

Click Next once you are done.
